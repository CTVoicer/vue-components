# CtDefaultImage

## Props

<!-- @vuese:CtDefaultImage:props:start -->

|src|Atributo `src` da imagem|`String`|`false`|-|

|width|Largura da imagem|`String` /  `Number`|`false`|150|
|height|Altura da imagem|`String` /  `Number`|`false`|150|
